    Description An optional DaylightSavingsTimeZone  
    Remarks It is optional, initially it is null.
In order to populate it a new tns.DaylightSavingsTimeZone object must be created and assigned to the property.
 
obj.DaylightSavingsTimeZone = new tns.DaylightSavingsTimeZone();
 
To 'un-populate' the property set it to null.

 
    Example 
cuahsiTimeSeries_v1_0Lib.TimeZoneInfo oParent = ...;

// The object is optional so initially is empty
System.Diagnostics.Debug.Assert(oParent.DaylightSavingsTimeZone == null);
oParent.DaylightSavingsTimeZone = new tns.DaylightSavingsTimeZone();
