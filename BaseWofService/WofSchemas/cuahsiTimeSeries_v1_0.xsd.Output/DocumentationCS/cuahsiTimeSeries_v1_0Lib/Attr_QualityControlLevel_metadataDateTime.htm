LiquidTechnologies.Runtime.Net20.XmlDateTime MetadataDateTime
  Property Description  
    Property Name MetadataDateTime  
    Property Type LiquidTechnologies.Runtime.Net20.XmlDateTime  
    Parent Class cuahsiTimeSeries_v1_0Lib.QualityControlLevel  
    XML Type Attribute  
    Accessors Read/Write  
    Description An optional LiquidTechnologies.Runtime.Net20.XmlDateTime  
    Remarks It is optional, initially it is not populated.
The property IsValidMetadataDateTime indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidMetadataDateTime = false.

 
    Also See IsValidMetadataDateTime - Indicates if the property has been populated.  
    Example 
cuahsiTimeSeries_v1_0Lib.QualityControlLevel oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidMetadataDateTime flag.
System.Diagnostics.Debug.Assert(oParent.IsValidMetadataDateTime == false);
      
// So it can be set as follows 
LiquidTechnologies.Runtime.Net20.XmlDateTime oNewItem = new LiquidTechnologies.Runtime.Net20.XmlDateTime();
oNewItem.SetDateTime(2005, 11, 29, 12, 56, 36);
oParent.MetadataDateTime = oNewItem;
      
// As the proprety is populated IsValidMetadataDateTime will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidMetadataDateTime == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.MetadataDateTime.ToString());
