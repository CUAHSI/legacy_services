Double Elevation_m
  Property Description  
    Property Name Elevation_m  
    Property Type Double  
    Parent Class tns.SiteInfoType  
    XML Type Element  
    Accessors Read/Write  
    Description An optional Double  
    Remarks It is optional, initially it is not populated.
The property IsValidElevation_m indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidElevation_m = false.

 
    Also See IsValidElevation_m - Indicates if the property has been populated.  
    Example 
tns.SiteInfoType oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidElevation_m flag.
System.Diagnostics.Debug.Assert(oParent.IsValidElevation_m == false);
      
// So it can be set as follows
oParent.Elevation_m = 1.5464575445;
      
// As the proprety is populated IsValidElevation_m will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidElevation_m == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.Elevation_m.ToString());
