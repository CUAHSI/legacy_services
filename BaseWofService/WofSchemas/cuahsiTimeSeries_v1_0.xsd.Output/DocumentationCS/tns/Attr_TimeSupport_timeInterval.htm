Int32 TimeInterval
  Property Description  
    Property Name TimeInterval  
    Property Type Int32  
    Parent Class tns.TimeSupport  
    XML Type Element  
    Accessors Read/Write  
    Description An optional Int32  
    Remarks It is optional, initially it is not populated.
The property IsValidTimeInterval indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidTimeInterval = false.

 
    Also See IsValidTimeInterval - Indicates if the property has been populated.  
    Example 
tns.TimeSupport oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidTimeInterval flag.
System.Diagnostics.Debug.Assert(oParent.IsValidTimeInterval == false);
      
// So it can be set as follows
oParent.TimeInterval = -1232154;
      
// As the proprety is populated IsValidTimeInterval will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidTimeInterval == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.TimeInterval.ToString());
