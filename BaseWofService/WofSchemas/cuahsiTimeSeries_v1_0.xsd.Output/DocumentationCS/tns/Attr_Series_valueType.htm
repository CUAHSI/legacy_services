tns.Enumerations.ValueTypeEnum ValueType
  Property Description  
    Property Name ValueType  
    Property Type tns.Enumerations.ValueTypeEnum  
    Parent Class tns.Series  
    XML Type Element  
    Accessors Read/Write  
    Description An optional ValueTypeEnum enumeration  
    Remarks It is optional, initially it is not populated.
The property IsValidValueType indicates if the property has been populated.
Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidValueType = false.


  
    Also See IsValidValueType - Indicates if the property has been populated.  
    Example 
ValueTypeEnum eItem;
tns.Series oParent = ...;

// The property is optional so is not initially set.
// attempts to read it will result in an exception being raised
System.Diagnostics.Debug.WriteLine(oParent.ValueType); // ERROR
      
// Before attempting to read the property you can test to see if ts valid
if (oParent.IsValidValueType)
{
  // its safe to read the value
  System.Diagnostics.Debug.WriteLine( oParent.ValueType);
}

System.Diagnostics.Debug.Assert( oParent.IsValidValueType == false);

// Setting the value, use any one of the following
oParent.ValueType = tns.Enumerations.ValueTypeEnum.FieldSpaceObservation;
oParent.ValueType = tns.Enumerations.ValueTypeEnum.Sample;
oParent.ValueType = tns.Enumerations.ValueTypeEnum.ModelSpaceSimulationSpaceResult;
oParent.ValueType = tns.Enumerations.ValueTypeEnum.DerivedSpaceValue;
oParent.ValueType = tns.Enumerations.ValueTypeEnum.Unknown;
System.Diagnostics.Debug.Assert( oParent.IsValidValueType == true);
