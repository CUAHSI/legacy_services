LiquidTechnologies.Runtime.Net20.XmlDateTime CreationTime
  Property Description  
    Property Name CreationTime  
    Property Type LiquidTechnologies.Runtime.Net20.XmlDateTime  
    Parent Class tns.QueryInfoType  
    XML Type Element  
    Accessors Read/Write  
    Description An optional LiquidTechnologies.Runtime.Net20.XmlDateTime  
    Remarks It is optional, initially it is not populated.
The property IsValidCreationTime indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidCreationTime = false.

 
    Also See IsValidCreationTime - Indicates if the property has been populated.  
    Example 
tns.QueryInfoType oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidCreationTime flag.
System.Diagnostics.Debug.Assert(oParent.IsValidCreationTime == false);
      
// So it can be set as follows 
LiquidTechnologies.Runtime.Net20.XmlDateTime oNewItem = new LiquidTechnologies.Runtime.Net20.XmlDateTime();
oNewItem.SetDateTime(2005, 11, 29, 12, 56, 36);
oParent.CreationTime = oNewItem;
      
// As the proprety is populated IsValidCreationTime will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidCreationTime == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.CreationTime.ToString());
