tns.Enumerations.SampleMediumEnum SampleMedium
  Property Description  
    Property Name SampleMedium  
    Property Type tns.Enumerations.SampleMediumEnum  
    Parent Class tns.VariableInfoType  
    XML Type Element  
    Accessors Read/Write  
    Description An optional SampleMediumEnum enumeration  
    Remarks It is optional, initially it is not populated.
The property IsValidSampleMedium indicates if the property has been populated.
Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidSampleMedium = false.


  
    Also See IsValidSampleMedium - Indicates if the property has been populated.  
    Example 
SampleMediumEnum eItem;
tns.VariableInfoType oParent = ...;

// The property is optional so is not initially set.
// attempts to read it will result in an exception being raised
System.Diagnostics.Debug.WriteLine(oParent.SampleMedium); // ERROR
      
// Before attempting to read the property you can test to see if ts valid
if (oParent.IsValidSampleMedium)
{
  // its safe to read the value
  System.Diagnostics.Debug.WriteLine( oParent.SampleMedium);
}

System.Diagnostics.Debug.Assert( oParent.IsValidSampleMedium == false);

// Setting the value, use any one of the following
oParent.SampleMedium = tns.Enumerations.SampleMediumEnum.SurfaceSpaceWater;
