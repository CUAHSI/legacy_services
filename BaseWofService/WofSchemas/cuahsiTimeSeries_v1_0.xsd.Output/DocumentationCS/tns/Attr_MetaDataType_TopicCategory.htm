String TopicCategory
  Property Description  
    Property Name TopicCategory  
    Property Type String  
    Parent Class tns.MetaDataType  
    XML Type Element  
    Accessors Read/Write  
    Description An optional String  
    Remarks It is optional, initially it is not populated.
The property IsValidTopicCategory indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidTopicCategory = false.

 
    Also See IsValidTopicCategory - Indicates if the property has been populated.  
    Example 
tns.MetaDataType oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidTopicCategory flag.
System.Diagnostics.Debug.Assert(oParent.IsValidTopicCategory == false);
      
// So it can be set as follows
oParent.TopicCategory = "Sample Value";
      
// As the proprety is populated IsValidTopicCategory will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidTopicCategory == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.TopicCategory.ToString());
