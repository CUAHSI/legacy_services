    Description An optional CensorCodeEnum enumeration  
    Remarks It is optional, initially it is not populated.
The property IsValidCensorCode indicates if the property has been populated.
Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidCensorCode = false.


  
    Also See IsValidCensorCode - Indicates if the property has been populated.
