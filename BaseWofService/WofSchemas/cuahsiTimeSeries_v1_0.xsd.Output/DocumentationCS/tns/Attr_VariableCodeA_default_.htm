Boolean default_
  Property Description  
    Property Name default_  
    Property Type Boolean  
    Parent Class tns.VariableCode  
    XML Type Attribute  
    Accessors Read/Write  
    Description An optional Boolean  
    Remarks It is optional, initially it is not populated.
The property IsValidDefault indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidDefault = false.

 
    Also See IsValidDefault - Indicates if the property has been populated.  
    Example 
tns.VariableCode oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidDefault flag.
System.Diagnostics.Debug.Assert(oParent.IsValidDefault == false);
      
// So it can be set as follows
oParent.default_ = false;
      
// As the proprety is populated IsValidDefault will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidDefault == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.default_.ToString());
