tns.Related Related
  Property Description  
    Property Name Related  
    Property Type tns.Related  
    Parent Class tns.VariableInfoType  
    XML Type Element  
    Accessors Read/Write  
    Description An optional Related  
    Remarks It is optional, initially it is null.
In order to populate it a new tns.Related object must be created and assigned to the property.
 
obj.Related = new tns.Related();
 
To 'un-populate' the property set it to null.

 
    Example 
tns.VariableInfoType oParent = ...;

// The object is optional so initially is empty
System.Diagnostics.Debug.Assert(oParent.Related == null);
oParent.Related = new tns.Related();
oParent.Related.[Some Property] = ...;
