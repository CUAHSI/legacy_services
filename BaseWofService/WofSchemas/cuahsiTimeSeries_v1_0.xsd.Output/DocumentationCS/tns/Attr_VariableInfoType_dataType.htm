tns.Enumerations.DataTypeEnum DataType
  Property Description  
    Property Name DataType  
    Property Type tns.Enumerations.DataTypeEnum  
    Parent Class tns.VariableInfoType  
    XML Type Element  
    Accessors Read/Write  
    Description An optional DataTypeEnum enumeration  
    Remarks It is optional, initially it is not populated.
The property IsValidDataType indicates if the property has been populated.
Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidDataType = false.


  
    Also See IsValidDataType - Indicates if the property has been populated.  
    Example 
DataTypeEnum eItem;
tns.VariableInfoType oParent = ...;

// The property is optional so is not initially set.
// attempts to read it will result in an exception being raised
System.Diagnostics.Debug.WriteLine(oParent.DataType); // ERROR
      
// Before attempting to read the property you can test to see if ts valid
if (oParent.IsValidDataType)
{
  // its safe to read the value
  System.Diagnostics.Debug.WriteLine( oParent.DataType);
}

System.Diagnostics.Debug.Assert( oParent.IsValidDataType == false);
