    Description An optional UnitsTypeEnum enumeration  
    Remarks It is optional, initially it is not populated.
The property IsValidUnitType indicates if the property has been populated.
Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidUnitType = false.


  
    Also See IsValidUnitType - Indicates if the property has been populated.
