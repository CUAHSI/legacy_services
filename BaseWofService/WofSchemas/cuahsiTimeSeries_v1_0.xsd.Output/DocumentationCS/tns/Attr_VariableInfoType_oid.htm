String Oid
  Property Description  
    Property Name Oid  
    Property Type String  
    Parent Class tns.VariableInfoType  
    XML Type Attribute  
    Accessors Read/Write  
    Description An optional String  
    Remarks It is optional, initially it is not populated.
The property IsValidOid indicates if the property has been populated. Any attempt to read a property that has not been populated will raise an exception.
To 'un-populate' the property SetValidOid = false.

 
    Also See IsValidOid - Indicates if the property has been populated.  
    Example 
tns.VariableInfoType oParent = ...;

// Because the property is Optional, it will NOT Initially be populated
// Attempting to read from it before it is set WILL RAISE AN EXCEPTION
// You can tell whether it is populated by checking 
// the IsValidOid flag.
System.Diagnostics.Debug.Assert(oParent.IsValidOid == false);
      
// So it can be set as follows
oParent.Oid = "Sample Value";
      
// As the proprety is populated IsValidOid will now return true
System.Diagnostics.Debug.Assert(oParent.IsValidOid == true);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine(oParent.Oid.ToString());
