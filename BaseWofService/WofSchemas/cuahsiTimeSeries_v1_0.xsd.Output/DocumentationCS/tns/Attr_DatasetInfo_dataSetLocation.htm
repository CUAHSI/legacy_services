tns.IGeogLocationType DataSetLocation
  Property Description  
    Property Name DataSetLocation  
    Property Type tns.IGeogLocationType  
    Parent Class tns.DatasetInfo  
    XML Type Element  
    Accessors Read/Write  
    Description An optional IGeogLocationType  
    Remarks It is optional, initially it is null.
The object type is an interface, this means that it must be populated using a class derived from IGeogLocationType.
In order to populate it, a new class deriving from tns.IGeogLocationType must be created and assigned to the property.
To 'un-populate' the property set it to null.
 
obj.DataSetLocation = new XXXX();
 
Where XXXX is one of the following classes
           GeogLocationType
           LatLonBox
           LatLonBoxType
           LatLonPoint
           LatLonPointType


  
    Example 
// One of the following
tns.GeogLocationType oItem = new tns.GeogLocationType();
tns.LatLonBox oItem = new tns.LatLonBox();
tns.LatLonBoxType oItem = new tns.LatLonBoxType();
tns.LatLonPoint oItem = new tns.LatLonPoint();
tns.LatLonPointType oItem = new tns.LatLonPointType();
tns.DatasetInfo oParent = ...;

// The property is optional, so it must be populated before it can be
// used.
oParent.DataSetLocation = oItem;

// Now its populated it can be used.
oParent.DataSetLocation.[Some Property] = ...;

// Note:
// It may be nessarsary to cast the class to its actaul type
// (see types listed above for posibles) in order to access 
// all of its properties i.e.
if (oParent.DataSetLocation[0].GetType().Name == "GeogLocationType")
{
  tns.GeogLocationType oDervItem;
  oDervItem = (tns.GeogLocationType)oParent.DataSetLocation[1];
  oDervItem.[Some Property] = ...;
}
if (oParent.DataSetLocation[0].GetType().Name == "LatLonBox")
{

}

{
