LiquidTechnologies.Runtime.Net20.ElementCol AnyElement
  Property Description  
    Property Name AnyElement  
    Property Type LiquidTechnologies.Runtime.Net20.ElementCol  
    Parent Class tns.Extension  
    XML Type Element  
    Accessors Read Only  
    Description A collection of untyped Element's  
    Remarks Only one Element within this class may be populated at a time (this collection, as a whole is counted as an element).
This collection may contain 0 or 0 to Many entries.
Emptying the collection allows a different element to be the populated one.
If there is already a populated item, and an item is added to this collection then an exception will be raised.


  
    Choice Properties Only one of the following properties may be populated at any one time
AnyElement

  
    Example 
LiquidTechnologies.Runtime.Net20.Element oElm;
tns.Extension oParent = ...;

// The collection object will already exist, but we may want to add items to it

// This method creates a new object adds it to the collection and 
// returns the new object
oElm = oParent.AnyElement.Add("PostCode", "http://www.liquid-technologies.com/Test");
oElm.Text = "SW5 R82";

// This way you can add an existing object to the collection
oElm = new LiquidTechnologies.Runtime.Net20.Element();
oElm.Name = "PostCode";
oElm.Namespace = "http://www.liquid-technologies.com/Test";
oElm.Text = "SW5 R82";
oParent.AnyElement.Add(oElm);

//-------------------------------------------------------------
// To iterate through the collection

// using the iterator
foreach(oElm in oParent.AnyElement)
{
  System.Diagnostics.Debug.WriteLine(oElm.Name);
}

// using an index
for(int i=0; i<oParent.AnyElement.Count; i++)
{
  oElm = oParent.AnyElement[i];
  System.Diagnostics.Debug.WriteLine(oElm.Name);
}
