LiquidTechnologies.Runtime.Net20.Element Extension
  Property Description  
    Property Name Extension  
    Property Type LiquidTechnologies.Runtime.Net20.Element  
    Parent Class tns.Series  
    XML Type Element  
    Accessors Read/Write  
    Description An Optional untyped Element  
    Remarks It is optional, initially it is null.
In order to populate it a new LtXmlLib.Element object must be created and assigned to the property.
To 'un-populate' the property set it to null.


  
    Example 
tns.Series oParent = ...;

// The object is optional so initially is empty
System.Diagnostics.Debug.Assert( oParent.Extension = null);
oParent.Extension = new LiquidTechnologies.Runtime.Net20.Element();
oParent.Extension.Name = "PostCode";
oParent.Extension.Namespace = "http://www.liquid-technologies.com/Test";
oParent.Extension.Text = "SW5 R82"
