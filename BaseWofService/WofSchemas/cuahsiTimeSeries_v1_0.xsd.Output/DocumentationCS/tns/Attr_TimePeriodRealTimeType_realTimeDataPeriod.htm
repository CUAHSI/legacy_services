LiquidTechnologies.Runtime.Net20.XmlDateTimeSpan RealTimeDataPeriod
  Property Description  
    Property Name RealTimeDataPeriod  
    Property Type LiquidTechnologies.Runtime.Net20.XmlDateTimeSpan  
    Parent Class tns.TimePeriodRealTimeType  
    XML Type Element  
    Accessors Read/Write  
    Description A mandatory LiquidTechnologies.Runtime.Net20.XmlDateTimeSpan  
    Remarks It is mandatory and therefore must be populated within the XML.
It is defaulted to [new LiquidTechnologies.Runtime.Net20.XmlDateTimeSpan()].

 
    Example 
tns.TimePeriodRealTimeType oParent = ...;

// Because the property is mandatory, it will already be populated
System.Diagnostics.Debug.Assert(oParent.RealTimeDataPeriod != null);
      
// So it can be read  as follows 
System.Diagnostics.Debug.WriteLine( oParent.RealTimeDataPeriod.ToString();

// So it can be set as follows 
oParent.RealTimeDataPeriod.SetDateTimeSpan(0, 0, 1, 0, 0, 0, 0, 0, 0);
      
// Or like this
LiquidTechnologies.Runtime.Net20.XmlDateTimeSpan oNewItem = new LiquidTechnologies.Runtime.Net20.XmlDateTimeSpan();
oNewItem.SetDateTimeSpan(0, 0, 1, 0, 0, 0, 0, 0, 0);
oParent.RealTimeDataPeriod = oNewItem;
